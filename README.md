Student Learning Engagement Tracking System

This is a web application built with Python (Flask) designed to track and manage student and employee engagement effectively. The system allows supervisors and managers to monitor tasks, reports, attendance, notifications, and scoring.

## Features
- Submit and track weekly plans
- Upload and manage documents, images, and videos
- Receive notifications on task updates
- Track employee attendance and punctuality
- Scoring system for task completion
- Generate and view reports dynamically
